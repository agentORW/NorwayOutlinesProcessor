# NorwayOutlinesProcessor
Processes the outline data for buildings and roads to be populated in Minecraft

## How to run
1. Download [Anaconda](https://www.anaconda.com/)
2. Open terminal in some directory.
3. Clone repository with `git clone https://github.com/agentORW/NorwayOutlinesProcessor.git`
4. Run `cd NorwayOutlinesProcessor`
5. Run `conda env create -f environment.yml`
6. Run `conda activate NOPenv`
7. Run `python -m ipykernel install --user --name=NOPenv`
8. Run `jupyter lab`
9. Make sure notebooks are connected to the "NOPenv" kernel.
